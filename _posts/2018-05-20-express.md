## 安装 express 命令行工具
>npm install -g express-generator

## 新建工程
>express -e blog
>cd blog
>npm install

## 运行
mac: DEBUG=blog node ./bin/www
window: 
>set DEBUG=blog "node  ./bin/www"
>npm start
